jquery 선택자

:fist : 선택자로 선택된 것들 중 0번 인덱스
:last : 선택자로 선택된 것들 중 마지막 인덱스
:odd : 선택자로 선택된 것들 중 홀수 인덱스에 있는 것
:even : 선택자로 선택된 것들 중 짝수 인덱스에 있는 것
:eq(index) : 매개변수와 일치하는 인덱스에 있는 것
:gt(index) : 매개변수보다 인덱스가 큰 것
:lt(index) : 매개변수보다 인덱스가 작은 것
